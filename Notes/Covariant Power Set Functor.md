``` ad-Definition
title: Definition.

Consider the category $\catset$. The <b>(covariant) power set functor on $\catset$</b> is the covariant functor $\ms{P}:\catset\to\catset$ sending each set $X$ to its power set $\pow\l(X\r)$ and each function $f:X\to Y$ to the direct image function $\im_f$.

```
