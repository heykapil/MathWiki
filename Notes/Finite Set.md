``` ad-Definition
title: Definition.

A set $X$ is said to be <b>finite</b> if $X\eqnum n$ for some $n\in\omega$.

```

<b>Remark.</b> Here, $n=\l\{0,\dots,n-1\r\}$ is interpreted set-theoretically.<span style="float:right;">$\blacklozenge$</span>
