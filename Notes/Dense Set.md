``` ad-Definition
title: Definition.

Let $\tpl{X,\mc{T}}$ be a topological space. A subset $S\subseteq X$ is said to be <b>dense in $X$</b> if $\bar{S}=X$; that is, if $U\cap S\neq\em$ for all neighborhoods $U$ of $x\in X$.

```
