``` ad-Definition
title: Definition.

Let $\tpl{X,<}$ be a totally ordered set. An <b>open ray in $X$ w.r.t. $<$</b> is a set of the form
$$\begin{equation}
    \l\{x\in X\st\ex a\in X:x>a\r\}\ \ \ \ \mathit{or}\ \ \ \ \l\{x\in X\st\ex b\in X:x<b\r\}.
\end{equation}$$
```
