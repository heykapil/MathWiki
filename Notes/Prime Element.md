``` ad-Definition
title: Definition.

Let $R$ be an integral domain. An element $p\in R$ is <b>prime</b> if $p\not\in R^\times$ and $p\divides\!\l(rs\r)$ implies either $p\divides r$ or $p\divides s$.

```
