``` ad-Definition
title: Definition.

Let $X$ be a set. A binary relation $<$ on $X$ is a <b>strict partial order on $X$</b> if $<$ is transitive and irreflexive on $X$.

```

<b>Remark.</b> Write $>\,\coloneqq\,<^{-1}$.<span style="float:right;">$\blacklozenge$</span>
