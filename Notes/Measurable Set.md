``` ad-Definition
title: Definition.

Let $\tpl{X,\mc{B},\mu}$ be a measure space. A subset $M\subseteq X$ is said to be <b>$\mu$-measurable</b> if $M=B\cup Z$ for some $B\in\mc{B}$ and some $\mu$-null set $Z\subseteq X$.

```
