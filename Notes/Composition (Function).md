``` ad-Definition
title: Definition.

Let $f:X\to Y$ and $g:Y'\to Z$ be functions with $Y\subseteq Y'$. The <b>composition of $g$ after $f$</b> is the function
$$\begin{equation}
    g\circ f:X\to Z\ \ \ \ \mathit{mapping}\ \ \ \ x\mapsto g\l(f\l(x\r)\r).
\end{equation}$$
```
